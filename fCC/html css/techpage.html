    Javascript

    Shortly named JS, Javascript is the main scripting language used with HTML. Since HTML wont perform algorithms, we need to use Javascript if we want to make our HTML pages less static and more dynamic for example

    JavaScript is a high-level, often just-in-time compiled language that conforms to the ECMAScript standard. It supports multi-paradigm, prototype based, object oriented and other possibilities;

    Cascading Style Sheets is way to stylize your HTML
